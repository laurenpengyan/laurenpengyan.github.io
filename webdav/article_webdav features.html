WebDAV Features

  1. Locking (also known as concurrency control), which prevents accidental overwriting of files;
  2. XML properties, which facilitate operations (such as storage and retrieval) on metadata, so that data about other data can be organized;
  3. The DAV protocol, which enables property setting, deleting, and retrieving;
  4. The DASL (DAV Searching and Locating) protocol, which enables searches based on property values for locating resources on the Web;
  5. Namespace manipulation, which supports copy and move operations. Collections, which are similar to file system directories, may be created and listed;
