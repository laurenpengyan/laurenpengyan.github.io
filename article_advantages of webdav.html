Advantages of WebDAV

  • Distributed Publishing of all content to the web: Users/workgroups can now use HTTP to directly publish their work to the Web. Workgroups can collaboratively author documents in-place on the web, using locking for concurrency control, to avoid overwrite problem.
  • WebDAV has no restrictions on the type of documents which can be authored. .
  • Managing repositories: WebDAV makes Web look like a large-grain network-accessible file system. WebDAV provides a standard interface to manage various repositories, over HTTP.
